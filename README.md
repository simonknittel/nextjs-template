## Features

- Next.js 15 + React 19
- Monorepo with npm workspaces and Turborepo
- TypeScript
- Prisma
- Tailwind CSS
- Authentication and authorization
  - Lucia Auth
  - Email and password
  - 2FA (TOTP)
- React Email
- GitHub Actions
- Renovate
- Playwright
- Vitest
- shadcn/ui
